Albaraa's React Portfolio (Vite)
==================================

Contents:
- Home, About, Projects, Education, Services, Contact pages (React + React Router)
- Simple NavBar with custom SVG logo
- Contact form captures data to localStorage and redirects to Home page
- Placeholder assets included (logo.svg, resume.pdf, project images)

How to run locally:
1. Ensure Node.js (v18+) and npm/yarn installed.
2. npm install
3. npm run dev
4. Open the shown local URL (http://localhost:5173 by default)

Deployment suggestions:
- Connect this repository to Vercel, Netlify, or Render and deploy (Vite is supported).
- Build: npm run build, then deploy the dist/ folder.

Git commit suggestions (example flow):
- Initial scaffold
- Add pages and routing
- Add contact form & localStorage capture
- Add images and resume PDF
- Final styling tweaks
